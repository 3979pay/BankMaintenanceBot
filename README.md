# BankMaintenanceBot

Bot Telegram dùng để gửi thông báo trạng thái ngân hàng.

## Chức năng

- Thông báo bảo trì
- Thông báo hoàn tất bảo trì
- Thông báo lỗi chuyển khoản
- Thông báo lỗi QR
- Menu chỉ dành cho admin

## Cài đặt

```bash
pip install -r requirements.txt
```

Tạo file `.env`

```
TELEGRAM_BOT_TOKEN=YOUR_BOT_TOKEN
```

Chạy bot

```bash
python bot.py
```
